##JS事件

| 事件        | 描述   |
| --------   | -----  |
| onclick     | 用户点击 HTML 元素 |
| onload        |  浏览器已完成页面的加载  | 
| onmouseover        |    用户在HTML元素上移过   |
| onmouseout        |    用户从一个HTML元素上移开鼠标 |
| onkeydown        |    用户按下键盘按键 |
| onchange |   HTML元素改变  |
